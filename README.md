# Triple Espresso

El proyecto consiste en una página web de una cafetería cuyo nombre es Triple Espresso.
En la página se encuentra un formulario básico para realizar reservas en el lugar. Además, se cuenta con los datos de contacto de la cafetería, sus horarios, ubicación y redes sociales.
Para el proyecto se utilizó HTML, los estilos para la página web en CSS, flexbox para la ubicación y disposición de los elementos y el formulario con todas sus características, para su correcto uso y diligenciamiento.

# Considero que algunas características y contenido adicional que se puede añadir a la página web son los productos que se ofrecen en la cafetería, es decir, como un catálogo, de manera que se tenga información detallada, que genere más interés por la cafetería.
